对java文件File类的操作

Posted on 2020-04-25

java.io.File 的每一个实例用于表示硬盘上的一个文件或者目录，通过File，我们可以进行如下操作：

  1. 访问一个文件或目录的属性信息，文件名，大小，修改日期，访问权限
  2. 创建文件，删除文件，创建目录，删除目录
  3. 访问目录中的文件或者子目录
  4. 不可以访问文件的数据，这点要特别注意。只能通过IO来访问文件数据
